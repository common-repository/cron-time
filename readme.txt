Cron Time plugin allows you to see all the set crons in WordPress.

== Description ==

You can see all the active cron in WordPress by using this plugin. 

Major features of this plugin include:

* Check active crons in WordPress
* Check hook name of different cron for development purposes
* Check time interval of hook
* Check minutes after which cron repeats itself

== Installation ==

* Upload the Cron Time plugin to the `/wp-content/plugins/` directory
* Activate the Cron Time plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to check active Crons ? =

* After activating plugin go to 'Cron Panel' menu is WordPress menu. It shows list of all crons active.
